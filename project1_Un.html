Project Overview

This experimental VR project explores how virtual spaces can evoke emotional responses through environmental storytelling and non-linear narrative structures. Players navigate fragmented memories represented as shifting architectural spaces.

Role

Lead Developer, Environment Designer, Sound Designer

Duration

6 months (Jan – Jun 2023)

Team

Solo project with mentor support

Development Journey

01

Concept & Research

Researched spatial memory theories and architectural phenomenology. Studied how physical spaces trigger emotional recall through materiality, lighting, and spatial proportions.

Key references: "The Poetics of Space" by Bachelard, memory palace techniques, and existing VR narrative experiences like "The Line" and "Dear Esther VR".

Concept Art
02

Prototyping & Mechanics

Built core interaction systems: object manipulation using XR Grab Interactables, teleportation locomotion with comfort vignette, and spatial audio triggers. Implemented procedural dissolution effect for transitioning between memory states.

Created modular room system allowing dynamic environment reconfiguration based on player's emotional state (measured via interaction patterns and dwelling time in spaces).

Prototypes
03

Environment Design

Designed five distinct memory environments: childhood bedroom, library, garden, hospital corridor, and abstract void space. Each environment uses specific color palettes, lighting schemes, and material properties to evoke different emotional tones.

Implemented dynamic lighting system responding to player position and interaction. Used Unity's URP with custom shaders for stylized rendering optimized for Quest 2 performance constraints.

Environment Design
04

Audio & Atmosphere

Designed layered soundscape using FMOD integration. Created 3D spatialized ambient loops, triggered sound effects, and subtle musical motifs that shift based on player location and story progression.

Recorded custom foley for object interactions to enhance tactile feel. Implemented reverb zones matching each environment's acoustic properties (tight reverb for bedroom, expansive for void space).

Sound Design
05

Testing & Iteration

Conducted 15 playtesting sessions with participants unfamiliar with VR and experienced VR users. Gathered feedback on comfort, narrative clarity, and emotional impact.

Major iterations: reduced locomotion speed by 30% to minimize motion sickness, added more environmental storytelling cues, simplified object interaction zones, and refined pacing between memory transitions.

Playtesting

Technical Implementation

XR Interaction System

Custom interaction framework built on Unity XR Interaction Toolkit. Implemented grab, push, and gesture-based interactions with haptic feedback calibrated for Quest 2 controllers.

Tools: C#, XR Toolkit, Oculus Integration SDK

Performance Optimization

Maintained steady 72fps on Quest 2 through aggressive LOD systems, occlusion culling, texture atlasing, and baked lighting. Draw calls kept under 120 per frame.

Techniques: URP optimization, mesh combining, shader LOD

Procedural Systems

Built dissolve shader using noise textures for memory fragmentation effect. Created scriptable object-based room configuration system for easy environment authoring.

Tools: Shader Graph, ScriptableObjects, Unity Events
